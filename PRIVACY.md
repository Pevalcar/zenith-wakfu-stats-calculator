# Política de Privacidad - Zenith Wakfu Stats Calculator

**Última actualización**: 22 de diciembre de 2025

## Recopilación de datos

Esta extensión **NO recopila, almacena ni transmite ningún tipo de dato personal**.

## Permisos utilizados

La extensión solicita los siguientes permisos:

### `storage`
- **Propósito**: Actualmente no se utiliza para guardar datos de usuario
- **Alcance**: Local al navegador del usuario
- **Nota**: Este permiso está reservado para futuras funcionalidades opcionales

### `host_permissions: zenithwakfu.com`
- **Propósito**: Permitir que la extensión inyecte código JavaScript en las páginas de zenithwakfu.com
- **Alcance**: Solo funciona en el dominio especificado
- **Uso**: Calcular y mostrar estadísticas de equipamiento en tiempo real

## ¿Qué hace la extensión?

1. **Lee el contenido de las tarjetas de equipamiento** en zenithwakfu.com
2. **Calcula sumas matemáticas** de las estadísticas mostradas
3. **Agrega elementos visuales** (tablas con resultados) a la página
4. **Todo el procesamiento es local** en tu navegador

## Sin seguimiento

- ❌ No utilizamos cookies
- ❌ No enviamos datos a servidores externos
- ❌ No realizamos seguimiento de comportamiento
- ❌ No compartimos información con terceros
- ❌ No mostramos publicidad

## Código abierto

El código fuente está disponible públicamente y puede ser auditado en cualquier momento.
